# Draco-IDE

Custom IDE for C++ development


**Overview**


Dark themed IDE that compiles and runs C++ code.


Contains tool for analyzing the time complexity of algorithms, nicknamed Fang.


**Interface**: C# / WPF

**Analysis Engine**: C++ (Linear Regression implementation with Feature Mapping)

**Interop:** C++ DLL communication with C# via P/Invoke

**Execution:** Process Redirection for compiling and capturing output
